## 提示 QQ 版本过低

- 尝试更换签名地址（kivi.config.toml 中）
- 可网络搜索"QQ 签名服务器"

## 提示网络异常

- 尝试切换同一网络（可使用代理）
- 尝试本地运行后，复制文件到服务器
- 尝试在 服务器 安装 官版 QQ/Tim 进行登录，再运行 KiviBot

## 服务器繁忙

Server is busy

- 尝试更改网络 DNS
- 119.29.29.29 / 114.114.114.114

## 群消息发送失败

- 账号被风控，需本地登录 QQ，手动在群内发送信息，解除限制。

## 上网环境异常

- 使用扫码登录
- 尝试切换同一网络(可使用代理)
- 尝试在 服务器 安装 官版 QQ/Tim 进行登录，再运行 KiviBot

## 提示密码已泄露

- 异地登录的原因，请打开设备锁。
